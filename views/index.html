About Shift Ninja

Shift Ninja aims to be a cross-platform mobile and web application (app). The overall concept of the app is to deliver a multifaceted system aimed at optimising the way in which Recruitment Agencies manage and distribute their available work vacancies.
A user friendly mobile app will enable agency recruits to view all work vacancies available to them. User’s e.g. the recruits will be able to bid or decline work opportunities through the mobile app. When recruitment staff receive a user’s bid request via the web app they will have the option to action it appropriately.
The mobile app will also have a calendar feature where they can set which days of the month they are available for work. This data will be shared with the recruitment agency.
Another important aspect of the mobile app is to provide users the ability to register their attendance at a work place by incorporating devices Near Field Communication (NFC) capabilities. This feature will work hand in hand with other key features to be discussed later in the proposal.
